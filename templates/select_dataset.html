      What is KDDCUP99

      The KDD Cup 1999 dataset is an early benchmark for intrusion detection systems. It covers various network attacks like DoS and U2R, with multiple attack formats represented. Features include protocol types, IP addresses, and attack labels. Split into training and testing files, it's a valuable resource for evaluating intrusion detection algorithms.

    • UNSW NB 15
      What is UNSWNB15

      The UNSW-NB15 dataset offers a wide range of network traffic scenarios, including 9 types of attacks like Fuzzers and DoS, alongside benign packets. It contains 49 features such as IP addresses, timestamps, and protocol types, extracted using tools like Argus and Bro-IDS. The dataset has two versions, one completely unbalanced with 2,540,044 records.
